# mvts-frontend-dashboard
Desarrollador Frontend (La "Cara" del Proyecto)

Herramientas: React, TypeScript, VS Code.

Misión: Esta persona no se preocupa por cómo llegan los datos al servidor. Su trabajo es armar el Dashboard, diseñar las tablas, poner el mapa en pantalla y hacer las peticiones fetch o axios hacia la API de Java para mostrar la información.

Hito de éxito: Tener la vista gerencial lista y mostrando los datos ficticios en pantalla.

Ventaja: Perfecto para quien tenga buen ojo para el diseño (UI/UX) y se sienta cómodo en el ecosistema web moderno.
